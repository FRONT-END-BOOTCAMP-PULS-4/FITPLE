## **기술 스택**

### **Front-end**
- **Next.js** (v15.2.4)
- **TypeScript**
- **Zustand** / **Context API** (상태관리)
- **Sass (CSS Module 기반)**  
- **CSS Variables**  
  - 컬러 / 마진 / 패딩 / 폰트 크기 등 디자인 시스템 반영  
- **Fetch API**
  - Next.js 자동 캐싱 기능을 활용하기 위해 `fetch` 사용 결정

### **Back-end**
- JavaScript 기반 (간단한 API 처리)

### **Database**
- **Supabase**

### **Design**
- **Figma**

### **OAuth 로그인**
- **Google**
- **Kakao**

### **협업 도구**
- GitHub
- Discord
- Notion
